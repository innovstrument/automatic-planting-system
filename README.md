# 自动种植系统
# 总论
植物种植系统，是一套可堆叠，可扩展的自动化植物培育系统。本系统旨在依靠人工光源和其它自动化设备实现较高水平的自动化种植。

## 第一节 系统结构和功能概览
本系统的控制，分为全局系统和本地控制系统。前者提供基本的服务，例如给水和电力配给。后者针对单一植物培养箱体进行控制。全局系统针对使用x86主机作为计算平台。对于局部控制系统，使用基于STM32的单片机作为控制平台。
